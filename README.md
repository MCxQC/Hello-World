* **Process Criteria Object**
  > Type: Object
  > - ProcessName
  > - ProcessPath

* **Function**
  > Type: String
  >
  > The name of the function to call when the event is created/terminated. To call the function append "_Created" and/or "_Terminated" to the function name.

* **Event Name**
  > Type: String
  > 
  > Name of the Event.
